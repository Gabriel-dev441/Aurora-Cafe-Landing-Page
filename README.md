#  Aurora Café

> Um site institucional moderno e responsivo, desenvolvido para apresentar o cardápio, o ambiente e a filosofia de uma cafeteria focada no aconchego e na qualidade.

---

##  Objetivo do Projeto
O principal objetivo é a divulgação da marca "Aurora Café", transmitindo digitalmente a mesma sensação de paz, tranquilidade e acolhimento que o espaço físico oferece. O público-alvo estratégico são pessoas que buscam um ambiente calmo para trabalhar (home office), estudar ou desfrutar de um bom café especial.

##  O Que a Empresa Oferece (Estrutura do Site)
* **Página Inicial (Hero):** Recepção impactante com chamada para ação (CTA) direta para o cardápio.
* **Seção Sobre:** Storytelling focado na filosofia de calmaria, grãos selecionados e conforto.
* **Cardápio Vitrine:** Opções da semana (Expressos, Cappuccinos, Bebidas Geladas e Doces) com descrições e preços visíveis.
* **Central de Atendimento:** Localização com a tag semântica `<address>`, horários diferenciados, botão para WhatsApp e links para redes sociais.

---

##  Destaques Técnicos
* **HTML5 Semântico:** Uso de `<header>`, `<main>`, `<section>`, `<article>` e `<footer>` para melhor SEO.
* **Acessibilidade:** Atributos `alt` descritivos e diretivas `aria-label` para leitores de tela.
* **Segurança da Informação:** Meta tags de proteção implementadas:
  * `Content-Security-Policy (CSP)` (anti-injeção de dados)
  * `X-Frame-Options: DENY` (anti-Clickjacking)
  * `X-Content-Type-Options: nosniff` (anti-MIME-sniffing)
* **Performance:** Integração otimizada com Google Fonts e Font Awesome via CDNs.

2. Abra a pasta do projeto no VS Code.
3. Clique com o botão direito no arquivo `index.html` e selecione **Open with Live Server** (ou dê um duplo clique no arquivo diretamente no seu gerenciador de arquivos).
